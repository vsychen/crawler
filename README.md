# Requisitos

  * Maven instalado.

# Instruções

  1. Abra o Prompt de Comando na pasta do projeto.
  2. Rode os comandos:
     1. mvn clean
	 2. mvn compile
	 3. mvn package
	 4. mvn exec:java -Dexec.mainClass="SpiderMain"
